# MERN Prescripteur App

Application de gestion des prescripteurs avec support multilingue (Arabe, Français, Anglais), import Excel et design responsive via TailwindCSS.

## Prérequis
- Node.js (v16+)
- MongoDB en cours d'exécution (localement via `mongodb://127.0.0.1:27017` ou URL distante)

## Installation Backend
1. `cd backend`
2. `npm install`
3. Copier `.env.example` vers `.env` et ajuster les variables.
4. `npm start` (ou `node server.js`)

## Installation Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` pour lancer l'application React sous Vite.
